Pengembangan Agen Prediksi Kesehatan Minuman

Model dan Algoritma

Aplikasi ini menggunakan algoritma Random Forest Classifier untuk memprediksi kesehatan minuman berdasarkan kandungan nutrisinya. Algoritma Random Forest bekerja dengan membuat banyak pohon keputusan dan menggabungkan hasilnya untuk mendapatkan prediksi yang lebih akurat.

Data dan Fitur

Model dilatih menggunakan data nutrisi yang mencakup fitur-fitur berikut:

  • Usia - Usia pengguna (tahun)
  • Jenis Kelamin - Pria atau Wanita
  • Takaran Saji (ml) - Volume per sajian
  • Jumlah Sajian per Kemasan - Jumlah porsi per kemasan
  • Energi Total (kcal) - Kalori total per sajian
  • Energi dari Lemak (kcal) - Kalori yang berasal dari lemak
  • Lemak Total (g) - Jumlah lemak total per sajian
  • Lemak Jenuh (g) - Jumlah lemak jenuh per sajian
  • Kolesterol (mg) - Jumlah kolesterol per sajian
  • Karbohidrat Total (g) - Jumlah karbohidrat total per sajian
  • Gula (g) - Jumlah gula per sajian
  • Garam (g) - Jumlah garam per sajian
  • Protein (g) - Jumlah protein per sajian

Fitur-fitur ini digunakan oleh model untuk menentukan apakah minuman yang dikonsumsi sehat atau tidak sehat.

Proses Pelatihan Model

Model dilatih dengan data yang berisi informasi nutrisi berbagai minuman. Label "Sehat" atau "Tidak Sehat" ditentukan berdasarkan aturan keseimbangan nutrisi. Model Random Forest dilatih dengan data ini untuk mengklasifikasikan minuman ke dalam dua kategori berdasarkan kandungan nutrisinya.

Evaluasi Kinerja

Model ini dievaluasi menggunakan metrik Accuracy dan Precision. Hasil evaluasi menunjukkan bahwa model memiliki akurasi yang cukup tinggi dalam memprediksi kesehatan minuman.

Unggah Data untuk Pengembangan

Jika Anda ingin mengembangkan atau melatih ulang model dengan data baru, silakan unggah file data nutrisi Anda (format: .csv atau .xlsx).
